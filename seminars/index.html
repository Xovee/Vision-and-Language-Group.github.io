Speaker:Junchen ZHU [PPT&Video(password: 1234)]
Topic:Cycle or not
Date:Sep 18, 2020
Abstract: Compared to supervised training in Image-to-Image translation addressed by GANs, unsupervised manner is a challenge where paired data are not available. To address this issue, cycle-like model is proposed to add an extra data flow from the target domain back to the source one. Many works based on cycle show impressive performances, which proves the good transferability of this design. However, as the models go deeper, the cyclic frameworks are getting harder to train, since the backward process paths are doubled in length by the twice domain translation. Also, the memory usage and computation FLOPS are nearly doubled blaming on the twice forward proceedings of generators. Therefore, a new kind of architecture without cycle design is proposed. To support the unsupervised training, the soft limitation as a query function in patch wise takes the place of the hard limitation by a simple L1 function in pixel wise. The experiments on various datasets verify the better performance of this no-cycle model, as well as the less time and computation resources cost.

Speaker:Qike Zhao [PPT&Video(password: 1234) ]
Topic:Inversion in GANs
Date:Sep 18, 2020
Abstract: Representation learning and generation capabilities are closely correlated in GANs. It is a belief that the ability to synthesize, or "create" observed data counting on understanding the pattern to make output and latent code invertible, which could effectively settle the problem of mode collapse.
In this presentation, I will introduce the basic framework and idea of GANs for junior students, and the representative works for solving inversion of images and latent code. Then the novel and recent work applied methods mentioned before would be introduced.
