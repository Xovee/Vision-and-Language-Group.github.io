Speaker:Ming Chen [PPT&Video(password: 1234)]
Topic:Scene Graph Generation
Date:Nov 20, 2020
Abstract: Scene graph generation is attracting more and more attention. Scene graph generation aims to generate a corresponding scene graph containing high-level semantic features for a given image, where vertices represent objects in the original image and edges represent the implicit relationship between them. It contains rich relational information and can be used for many downstream tasks, such as VQA and image captioning. I will share two papers, one of which is to realize the scene graph generation by introducing global context information, and the other is to propose a counterfactual reasoning method to eliminate the bias generated during model training.

Speaker:Lanxin Miao [PPT&Video(password: 1234) ]
Topic:Attention mechanism in image-text matching
Date:Nov 20, 2020
Abstract: As a typical cross-modal problem, image-text bidirectional retrieval relies heavily on the joint embedding learning and similarity measure for each image-text pair. It remains challenging because prior works seldom explore semantic correspondences between modalities and semantic correlations in a single modality at the same time. Besides, existing methods mainly explore object-based alignment between image objects and text words, while ingnore the high-order semantic information. To address the above issues, attention mechanism has been introduced. Two papers related to this area will be introduced in this presentation.
